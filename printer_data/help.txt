/joke - send a joke
/joke baneks - send joke from a 'B' category
/joke add <text> - add a joke
/joke search <text> - search for jokes in database
/joke i <num> - send joke number <num>
/joke delete <num> - delete joke number <num>
/joke hivemind {on|off|} - turn hivemind on or off; display current state
/whattimeisit - display current time
/ping - say "Pong!"
/hm - say "Hm."
/id - show your id (or id of user you replying are)
/user <id> show admin - check if user is admin
/user <id> set admin {true|false} - set user admin status
/user <id> add group <group> - add user to group
/user <id> delete group <group> - delete user from group
/user <id> show groups - show user groups
/gitversion - display last commit id
/help - display this help
